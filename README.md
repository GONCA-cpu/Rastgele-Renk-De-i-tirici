 Rastgele Renk Değiştirici / Random Background Color Changer

 Türkçe

Bu küçük JavaScript projesi, kullanıcı etkileşimi ile sayfanın arka plan rengini rastgele değiştirmeyi sağlar.  
Projede şu özellikler vardır:

- Bir butona tıklayınca arka plan renginin rastgele değişmesi  
- Tıklama sayısının sayılması ve ekranda gösterilmesi  
- Basit HTML, CSS ve JavaScript kullanımı  
- DOM manipülasyonu ve event listener uygulamaları  

Bu proje başlangıç seviyesindeki JavaScript öğrenenler için uygundur ve temel JS kavramlarını uygulamalı olarak pekiştirmek için tasarlanmıştır.

 English

This small JavaScript project allows the user to change the background color of the page randomly via interaction.  
Features of the project include:

- Randomly changing the background color when a button is clicked  
- Counting the number of button clicks and displaying it on the screen  
- Using basic HTML, CSS, and JavaScript  
- Practicing DOM manipulation and event listener usage  

This project is suitable for beginner JavaScript learners and is designed to reinforce basic JS concepts in a practical way.
